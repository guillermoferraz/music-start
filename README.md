Application created with nodejs, handelbars and mongodb as a database.
Crud operations, create, read, update and delete data.
Authentication and protection of routes using passport, passport-local and bcryptjs to encrypt the password.

It's about saving your favorite videos and favorite online radios.
Data rendering using apgination with mongodb and bootstrap.

You can try it at https://music-start.herokuapp.com

Installation.
npm install
node src / index.js
